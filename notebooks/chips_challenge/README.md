# Getting Started Tutorial

## Installation Instructions

If you have not installed all required dependencies, follow the [Installation Guide](https://openvinotoolkit.github.io/anomalib/getting_started/installation/index.html).
You will also need OpenVINO, so make sure you install that option.

## Notebook Contents

This notebook demonstrates usage of anomaly detection on a real world use case of anomaly detection on pharmaceutical objects.
We use Sensum Solid Oral Dosage Forms (Sensum SODF) dataset that contains two forms of solid oral dosage forms.
We use only first form of capsule.

Notebook contains the following sections:

- Installing Anomalib
- Configuration
- Dataset Preparation
- Model Preparation
- Model Training and Validation
- OpenVINO Inference
